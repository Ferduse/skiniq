# SkinIQ

An AI-powered skincare app that analyzes ingredients and builds personalized routines.

## What it does

SkinIQ helps anyone understand what's in their skincare products and whether those products are safe to use together.

## Tech Stack
- **Backend** — Python, FastAPI, OpenAI API, ChromaDB, SQLModel
